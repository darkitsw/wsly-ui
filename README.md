# WSLY

![build](https://img.shields.io/badge/build-building-yellow.svg) ![license](https://img.shields.io/packagist/l/doctrine/orm.svg)
> A Vue.js 2.0 UI Toolkit for Web.

## Install

```shell
npm install wsly-ui -S
```

### Quick Start

```js
import Vue from 'vue'
import WSLY from 'wsly-ui'

Vue.use(WSLY)
```
